📦 INSTALLATION
═══════════════════════════════════════════════════════════════

1. Copy this folder (ZenAlign) to:
   World of Warcraft\Interface\AddOns\

2. Launch WoW and enable addon at character select

3. Enter game - you should see:
   "ZenAlign: v1.0.0 loaded. Type /zenalign for options."


🎮 BASIC USAGE (30 SECONDS)
═══════════════════════════════════════════════════════════════

1. Show the grid:
   /zenalign grid

2. Open main window:
   /zenalign

3. Move a frame:
   - Click "Player Frame" in the list
   - Drag the green box that appears
   - Release - it SNAPS to the grid! ✨
   - Right-click green box to close


⌨️ COMMANDS
═══════════════════════════════════════════════════════════════

/zenalign           Open main window
/zenalign grid      Toggle grid display
/zenalign snap      Toggle grid snapping
/zenalign help      Show all commands
/za                 Short alias


🎚️ SETTINGS
═══════════════════════════════════════════════════════════════

In the main window (/zenalign):

☑ Show Grid             Toggle grid on/off
☑ Enable Grid Snapping  Snap frames to grid
☑ Show Coordinates      Show cursor position

Grid Size: [8-128]      Adjust grid spacing
                        (32 pixels = default)

💡 TIPS
═══════════════════════════════════════════════════════════════

• Start with 32px grid (default) - good balance
• Use 8-16px for pixel-perfect precision
• Use 64px for quick rough positioning
• Enable grid before moving for visual reference
• Positions save automatically!


🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Grid not showing?
  → Try /reload
  → Check "Show Grid" is checked in /zenalign

Frame won't move?
  → Some frames need to exist first (target something)
  → Exit combat if protected frame

Snapping not working?
  → Enable "Grid Snapping" checkbox
  → Grid must be enabled (uses grid size)

Positions not saving?
  → Type /reload to force save
  → Check WoW has write access to WTF folder
